# Customer-Churn-Prediction

This repository contains an end-to-end Machine Learning project to analyze and predict customer churn. Customer churn occurs when customers stop doing business with a company. Identifying these customers early allows businesses to take proactive retention measures.

---

## Project Stack & Tools
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn
* **Machine Learning Algorithms:** 
  * XGBoost
  * CatBoost
  * LightGBM
* **Environment:** Jupyter Notebook / Google Colab

---

##  Project Structure
* `ChurnZero_SCodes_Code.ipynb`: The complete project code containing data preprocessing, feature engineering, model training, and evaluation.
* `ChurnZero_SCodes_Predictions.csv`: The final output file containing the churn prediction results.
* `README.md`: Project documentation (this file).

---

## 🚀 Key Objectives
1. Perform Exploratory Data Analysis (EDA) to find key reasons behind customer churn.
2. Handle class imbalance and preprocess data for machine learning.
3. Train and tune high-performance boosting algorithms (XGBoost, CatBoost, LightGBM).
4. Evaluate models using metrics like Accuracy, Precision, Recall, and F1-Score.
